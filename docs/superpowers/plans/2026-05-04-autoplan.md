# Autoplan Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Automatically run `terraform plan` for affected projects when a PR is opened or updated, without requiring a `/tf plan` comment.

**Architecture:** A new `tf-autoplan.yml` workflow triggers on `pull_request: [opened, synchronize, reopened]` and calls the existing JS action with a new `trigger: autoplan` input, bypassing command parsing. The JS action filters projects by a new per-project `autoplan.enabled` flag (defaults to `true`) and resolves affected projects via changed-file matching — the same logic used by manual `/tf plan`. Matching projects proceed directly into the existing `tf-plan.yml` reusable workflow.

**Tech Stack:** JavaScript (Node 24), `@actions/core`, `@actions/github`, GitHub Actions YAML, Jest

- [ ] **Step 3: Update `src/main.js` — read `trigger` and add autoplan dispatch**

Replace the beginning of `run()` up through the command parse guard:

```js
async function run() {
  try {
    const token = core.getInput('github-token', { required: true })
    const configPath = core.getInput('config-path') || '.terraform-deployment'
    const trigger = core.getInput('trigger') || 'comment'
    const commentBody = core.getInput('comment-body') || ''
    const prNumber = parseInt(core.getInput('pr-number', { required: true }), 10)
    const headSha = core.getInput('head-sha', { required: true })
    const changedFiles = JSON.parse(core.getInput('changed-files', { required: true }))
    const codeownersContent = core.getInput('codeowners-content') || ''
    const approvedReviewers = JSON.parse(core.getInput('approved-reviewers') || '[]')
    const commenter = core.getInput('commenter', { required: true })

    const octokit = github.getOctokit(token)
    const { owner, repo } = github.context.repo

    // 1. Load and validate config (needed by both triggers)
    let config
    try {
      config = loadConfig(configPath)
    } catch (err) {
      if (trigger === 'comment') {
        await postComment(octokit, owner, repo, prNumber, renderConfigError(err.message))
      }
      core.setOutput('action', 'none')
      return
    }

    // 2. Autoplan path: skip command parsing, filter by autoplan.enabled, resolve by changed files
    if (trigger === 'autoplan') {
      const eligibleRaw = (config.projects || []).filter(
        (p) => p.autoplan?.enabled !== false
      )
      const rawChanged = resolveChangedProjects(changedFiles, eligibleRaw)
      if (rawChanged.length === 0) {
        core.setOutput('action', 'none')
        return
      }
      const targetProjects = rawChanged.map((p) => mergeWithDefaults(config.defaults || {}, p))
      const payload = buildPlanPayload(targetProjects)
      await postComment(octokit, owner, repo, prNumber, renderPlanQueued(targetProjects))
      core.setOutput('action', 'plan')
      core.setOutput('projects', JSON.stringify(payload))
      core.setOutput('pr-number', String(prNumber))
      core.setOutput('head-sha', headSha)
      core.setOutput('commenter', commenter)
      return
    }

    // 3. Comment path: parse the /tf command
    const parsed = parseCommand(commentBody)
    if (!parsed.valid) {
      await postComment(octokit, owner, repo, prNumber, renderCommandError(parsed.error))
      core.setOutput('action', 'none')
      return
    }
```

### Task 5: Create `tf-autoplan.yml` in the action repo and template

**Files:**

- Create: `template/.github/workflows/tf-autoplan.yml`

- [ ] **Step 1: Create `.github/workflows/tf-autoplan.yml`**

```yaml
name: Terraform Auto-Plan

on:
  pull_request:
    types: [opened, synchronize, reopened]

permissions:
  contents: read
  pull-requests: write
  issues: write
  statuses: write
  id-token: write
  actions: read

concurrency:
  group: tf-${{ github.event.pull_request.number }}
  cancel-in-progress: false

jobs:
  dispatch:
    runs-on: ubuntu-latest
    outputs:
      action: ${{ steps.control.outputs.action }}
      projects: ${{ steps.control.outputs.projects }}
      pr-number: ${{ steps.control.outputs.pr-number }}
      head-sha: ${{ steps.control.outputs.head-sha }}
      commenter: ${{ steps.control.outputs.commenter }}

    steps:
      - name: Check out repository
        uses: actions/checkout@v4
        with:
          ref: ${{ github.event.pull_request.head.sha }}

      - name: Gather PR changed files
        id: pr-meta
        uses: actions/github-script@v7
        with:
          script: |
            const files = await github.paginate(github.rest.pulls.listFiles, {
              owner: context.repo.owner,
              repo: context.repo.repo,
              pull_number: context.payload.pull_request.number,
            })
            core.setOutput('changed-files', JSON.stringify(files.map(f => f.filename)))

      - name: Run control action (autoplan)
        id: control
        uses: ./
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          config-path: .terraform-deployment
          trigger: autoplan
          pr-number: ${{ github.event.pull_request.number }}
          head-sha: ${{ github.event.pull_request.head.sha }}
          changed-files: ${{ steps.pr-meta.outputs.changed-files }}
          commenter: ${{ github.event.pull_request.user.login }}

  plan:
    needs: dispatch
    if: needs.dispatch.outputs.action == 'plan'
    uses: ./.github/workflows/tf-plan.yml
    with:
      projects: ${{ needs.dispatch.outputs.projects }}
      pr-number: ${{ needs.dispatch.outputs.pr-number }}
      head-sha: ${{ needs.dispatch.outputs.head-sha }}
      commenter: ${{ needs.dispatch.outputs.commenter }}
    secrets: inherit
```
